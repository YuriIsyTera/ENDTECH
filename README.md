# ENDTECH

An expansion pack built upon Nova Engineering: World, adding a large number of machines to optimize vanilla content, along with plenty of brand-new original content.

> 一个基于《新星工程：世界》制作的拓展包，添加了大量用于优化原版内容的机器以及众多全新的原创内容。

# 欢迎游玩终末工程 ENDTECH

本拓展包旨在增添后期游玩流程，并为玩家提供多类可建造巨构与机制，丰富游玩体验。
